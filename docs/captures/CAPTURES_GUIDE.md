# Captures guide

Place README screenshots in this folder with the exact names below:

1. 01-dashboard.png
2. 02-import.png
3. 03-ibnr.png
4. 04-bilan-history.png
5. 05-audit.png
6. 06-exports.png

## Important

README currently displays placeholder SVG files from [placeholders](placeholders)
so the gallery always renders professionally.

To switch to real screenshots:

1. Add PNG files using the exact names above.
2. Update the captures section in [README.md](../../README.md) to use the PNG paths.

## Recommended format

- PNG
- Width: 1600 px to 2200 px
- Keep browser zoom at 100%
- Hide personal/sensitive data before capture

## Quick capture flow on Windows

1. Open page in browser.
2. Use Win + Shift + S.
3. Save image with one of the exact filenames above.
4. Put files in this folder.
5. Commit and push.
